# 🇮🇳 India Analytics Dashboard

A comprehensive data analytics dashboard analyzing India's demographics, economy, and future predictions across 36 states and 640 districts.

## 🚀 Live Demo

[View Live Dashboard](https://your-app-url.streamlit.app)

## ✨ Features

- 🏠 **Interactive Home Page** with key metrics
- 🗺️ **State Overview** - Visualize all states
- 📚 **Literacy Analysis** - Education metrics
- 👫 **Gender Analysis** - Sex ratio insights
- 🕊️ **Religious Diversity** - Religion breakdown
- 💼 **Economic Indicators** - Per capita income
- 🌐 **Digital India** - Internet penetration
- 🏙️ **Urban vs Rural** - Urbanization analysis
- 🔍 **Hidden Patterns** - Surprising correlations
- 📊 **State Comparison** - Compare 2-5 states
- 🏘️ **District Drill Down** - 640 districts
- ⏰ **Time Travel** - 2011 to 2030 slider
- 📈 **2011 vs 2024** - Decade comparison
- 🔮 **2030 Predictions** - Future forecasts
- 🤖 **AI Insights** - Smart analysis
- 📄 **Download Reports** - Export data

## 📊 Data Sources

- Census of India 2011 (640 districts)
- Real 2024 data from UN & Government
- NFHS-5 (2019-2021)
- RBI Economic Statistics

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- NumPy
